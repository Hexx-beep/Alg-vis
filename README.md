# Algorithm Visualizer

A comprehensive web application for visualizing and understanding various algorithms and data structures through interactive visualizations. Built with modern web technologies to provide an intuitive and educational experience for both beginners and advanced users.

![Algorithm Visualizer](https://via.placeholder.com/1200x600/1e293b/ffffff?text=Algorithm+Visualizer+Dashboard)

## üöÄ Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, or Edge)
- Node.js (for development)
- npm or yarn (for development)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/algorithm-visualizer.git
   cd algorithm-visualizer
   ```

2. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   ```

3. Start the development server:
   ```bash
   npm start
   # or
   yarn start
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## üìö How to Use

1. **Graph Visualization**
   - Add nodes by clicking on the canvas
   - Connect nodes by dragging from one node to another
   - Right-click on nodes/edges for more options
   - Select an algorithm from the sidebar to visualize

2. **Sorting Visualizer**
   - Choose a sorting algorithm
   - Adjust array size and speed
   - Generate new arrays or use custom input
   - Compare multiple algorithms side by side

3. **Search Visualizer**
   - Select a search algorithm
   - Set the target value
   - Step through the search process
   - Compare different search strategies

4. **Tree Visualizer**
   - Create custom trees or use presets
   - Perform operations (insert/delete/search)
   - Visualize different traversal methods
   - See balance factors and rotations

## Usage

### Graph Controls
- **Add Node**: Click anywhere on the canvas to add a node
- **Add Edge**: Click on a node, then click on another node to create an edge
- **Delete Node/Edge**: Right-click on a node or edge and select "Delete"
- **Generate Graph**: Use the sidebar to generate various types of graphs
- **Clear Graph**: Remove all nodes and edges
- **Save/Load**: Save your current graph or load a previous one

### Running Algorithms
1. Select an algorithm from the sidebar
2. Configure any required parameters
3. Click "Run" to visualize the algorithm
4. Use the "Step" button to go through the algorithm step by step
